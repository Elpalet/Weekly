# Weekly

Private Weekly Skill: 个人科研周报写作。

一个用于按个人科研周报风格整理本周进展、问题与下周计划的 Codex skill。

## 安装

将 `personal-weekly-report` 文件夹放入本地 Codex skills 目录：

```text
~/.codex/skills/personal-weekly-report
```

重启 Codex 后使用：

```text
使用 $personal-weekly-report 根据我这周的工作记录写一份周报，并输出 DOCX。
```

## 内容

- `SKILL.md`：触发条件与写作流程
- `references/writing-style.md`：周报结构和语言习惯
- `references/docx-format.md`：DOCX 页面及版式规则
- `agents/openai.yaml`：Codex 展示元数据

本仓库不包含历史周报原文或分析数据。
